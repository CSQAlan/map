## 几何生成规则

- 节点坐标是唯一校准源。
- POI 根据 `linked_node_code` 吸附到对应节点。
- 路段 LineString 由起点和终点自动生成，不再维护重复 WKT。
- 高德前端只消费后端返回的 GCJ-02，禁止二次转换。

## 仍需现场复核

照片 GPS 能纠正整体偏移，但手机单点定位和拍摄位置不等同于道路中心线。比赛前应在高德底图上逐一复核入口、荷塘、楼栋、内部主路和北侧林下步道；若折线穿过建筑或水面，应补采轨迹点，而不是移动照片坐标来掩盖问题。

下一轮采集建议使用手机连续轨迹，每 3 至 5 米记录一点，并记录定位精度。轨迹审核通过后再替换当前“端点直连”的路段几何。
